東京圏輸送情報システム

運営体制強化に伴う補助要員の募集について
重要機密：学術・技術研究用資料

現在運用中の「東京圏輸送情報システム」において、提供データの更なる高精度化およびシステム稼働率の向上を目的とし、運営業務を補助する要員を募集いたします。 本システムはODPT API等を基盤とした公共交通情報の統合モニタリングを行っており、これらインフラの保守・管理に寄与いただける方の参画を募ります。

職務内容（Role）

  • システムデバッグ： 既存プログラムの瑕疵特定および修正案の策定
  • データインテグリティ検証： 配信データの整合性確認および異常値の検知
  • インフラ監視補助： Ubuntu Linuxを用いたサーバーの稼働状況の常時監視
  • 機能拡張開発： 新規モジュールの設計、Pythonを用いたAPI連携の高度化

採用技術スタック（Environment）

Backend/Infrastructure:
Python 3.1x (Core Logic) Ubuntu Server (Deployment) Cloudflare Workers (Edge Computing) GitHub Actions (CI/CD Pipeline)

Frontend/Data:
HTML5 / CSS3 (Modern UI) JavaScript (ES6+) JSON / GTFS Data Structures GeoSpatial Visualization (Leaflet.js)

External Services:
Public Transportation Open Data API Discord API (Webhook/Bot Notification) Git / GitHub Version Control

応募要件（Requirement）

  • 鉄道輸送、または公共交通データの構造・運用に深い関心を有すること
  • 機密保持の徹底： プロジェクト内部で得た情報を外部に漏洩させない倫理観
  • チャットツール（Discord等）を用いた、論理的かつ円滑な連携が可能であること
