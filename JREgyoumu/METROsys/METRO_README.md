# METRO OPERATIONS

東京メトロ9路線を対象にした、単一HTMLの業務情報ポータルです。

## 開き方

`metro.html` をブラウザで開いてください。ODPTアクセストークンはHTML内に設定済みです。

## 使用データ

- 運行情報（TrainInformation）
- 路線系統情報（Railway）
- 駅情報（Station）
- 列車時刻表（TrainTimetable）
- 駅時刻表（StationTimetable）
- 運賃情報（RailwayFare）
- 乗降者数情報（PassengerSurvey）
- 鉄道関連リアルタイム情報（GTFS-RT Alert）
- 鉄道関連情報（GTFS/GTFS-JP）
- 画像情報（公開データ接続一覧）

列車の「計画位置」は列車時刻表と現在時刻から算出した推定で、実測位置ではありません。GTFS-RTは東京メトロの配信仕様どおり運行アラートとして使用します。

ダッシュボードと路線運行画面では、選択路線の全駅と現在時刻に該当する列車を路線図上へ表示します。架空の列車やデモ値は表示せず、該当データがない場合は「在線列車なし」と表示します。

重いデータは画面操作に応じて必要な路線・駅・区間だけ取得します。アクセストークン入りのHTMLを第三者へ渡したり、公開サーバーへ配置したりしないでください。
