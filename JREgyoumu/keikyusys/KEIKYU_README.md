# 京急電鉄情報ポータル

`keikyu.html` をブラウザで開くと使用できます。追加サーバーやインストールは不要です。

## 表示内容

- 京急本線・空港線・大師線・久里浜線・逗子線の路線別在線モニタ
- ODPT の実列車ロケーションに基づく駅・駅間の在線表示
- 京急線の運行情報と路線別在線本数
- 列車番号、種別、方向、在線位置、行先、遅延の一覧
- 60秒ごとの自動更新と手動更新

## データ上の注意

- 品川—泉岳寺間は列車ロケーション情報の配信対象外です。
- 運行情報は京急線全体の1件として配信されるため、各路線カードには同じ全体情報を表示します。
- APIから実データを取得できない場合、架空列車やデモデータは表示しません。
- Challenge 2026アクセストークンは、依頼どおりHTML内に設定しています。

データ提供: 公共交通オープンデータセンター / 京急電鉄
