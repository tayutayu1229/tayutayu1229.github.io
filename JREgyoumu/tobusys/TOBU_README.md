# TOBU TRAFFIC CONTROL

東武鉄道の路線別在線モニタを中心とした、単一HTMLの業務情報ポータルです。

## 起動

`tobu.html` をブラウザで開いてください。在線は60秒ごとに自動更新されます。

## データと認証

東武鉄道のデータは `https://api-challenge.odpt.org/` で提供される「公共交通オープンデータチャレンジ2026」限定データです。通常のODPT APIトークンとは別に、Challenge 2026用アクセストークンが必要です。

`tobu.html` 内の次の定数にトークンを設定します。

```javascript
const TOBU_CHALLENGE_TOKEN='Challenge 2026用トークン';
```

Challenge 2026用の既定トークンをHTMLへ設定済みです。認証や配信に問題がある場合も、架空在線やデモデータは表示しません。

## 在線モニタ

- ODPT `odpt:Train` の `odpt:fromStation` と `odpt:toStation` を駅間在線として表示
- 順方向と逆方向を複線で分離
- 列車番号、種別、行先、遅延を列車マーカーへ表示
- 路線ごとの運行情報を同じ画面へ表示
- 60秒自動更新と手動更新

館林以北の各線、新栃木以北の各線は列車ロケーション情報の対象外です。亀戸線の小村井―東あずま―亀戸水神間は、配信仕様上、在線位置を区別できません。
