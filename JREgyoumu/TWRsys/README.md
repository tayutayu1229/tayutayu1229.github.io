# RINKAI OPERATIONS

東京臨海高速鉄道（りんかい線）の業務情報ポータルを想定した、単一HTMLのデモサイトです。

## 開き方

`index.html` をブラウザで開くと、そのまま表示できます。

ODPTアクセストークンは `index.html` 内に設定済みです。ファイルを開くと、運行情報と駅時刻表を自動取得します。

列車の実測位置情報は使用していません。駅時刻表の列車番号と各駅の発車時刻を突き合わせ、現在時刻における計画上の駅間位置を推定表示します。

トークン入りのHTMLを第三者に渡したり、公開サーバーへ配置したりしないでください。

## 使用データ

- 東京臨海高速鉄道 運行情報（TrainInformation）
- 東京臨海高速鉄道 駅情報（Station）
- 東京臨海高速鉄道 駅時刻表（StationTimetable）
- 東京臨海高速鉄道 路線系統情報（Railway）

出典：公共交通オープンデータセンター

> 運行情報と駅時刻表はODPTの実データです。列車位置は時刻表による計画推定であり、実際の位置や遅延を示すものではありません。乗降者数など一部項目には参考値が含まれます。
