# RINKAI OPERATIONS

東京臨海高速鉄道（りんかい線）の業務情報ポータルを想定した、単一HTMLのデモサイトです。

## 開き方

`index.html` をブラウザで開くと、そのまま表示できます。

ODPTの実データを起動時から取得する場合は、`index.html` の末尾付近にある次の行へ、開発者サイトで発行されたアクセストークンを貼り付けてください。

```javascript
const ODPT_ACCESS_TOKEN = 'ここにアクセストークンを貼り付け';
```

空欄のままでも画面の「設定」から一時入力できます。トークン入りのHTMLを第三者に渡したり、公開サーバーへ配置したりしないでください。

## 使用データ

- 東京臨海高速鉄道 運行情報（TrainInformation）
- 東京臨海高速鉄道 駅情報（Station）
- 東京臨海高速鉄道 駅時刻表（StationTimetable）
- 東京臨海高速鉄道 路線系統情報（Railway）

出典：公共交通オープンデータセンター

> 本サイトは業務画面のデザイン・操作デモです。表示中の列車位置、乗降者数、時刻等にはデモ値が含まれます。実運用時は認証情報をサーバー側で管理し、ODPT APIレスポンスを中継する構成を推奨します。
